The Challenge

Build something cool with your team's API, and deploy it to Heroku. We'll present on Monday.

Submitting

  • Try to work with your group. If you can't, build your own.
  • These don't need to be crazy complex! Just make something simple that works with your API.
  • To submit, create a text file with your group member names, and the URLs of your app's GitHub repo and deployed location. Put that in our class repo folder for each group member.

Mixins

Need more firepower? Use additional APIs to get more data! Any API you can find is fair game for supporting your primary challenge API.

  • Google Image search (for supporting images)
  • Giphy search (for animated GIFs)
  • Alchemy (performs sentiment analysis of text)
  • Yoda Speak (formats text as Yoda would say it)
  • StubHub (event ticket sales)
  • This for That (their api offers random nouns)
